🎨 CSS DASAR (CASCADING STYLE SHEETS)

CSS adalah bahasa yang digunakan untuk mengatur tampilan website seperti warna, ukuran, layout, dan desain. CSS membuat website menjadi lebih menarik dan tidak polos seperti HTML saja.

🧠 1. Pengertian CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur style atau tampilan dari HTML. CSS tidak mengatur struktur, hanya mengatur desain.

🎯 2. Fungsi CSS

  • Mengatur warna teks
  • Mengatur background
  • Mengatur ukuran teks
  • Mengatur layout halaman
  • Membuat website lebih menarik

🧾 3. Cara Penulisan CSS

Ada 3 cara:

  • Inline CSS
  • Internal CSS
  • External CSS
p {
  color: red;
}

🎨 4. Selector CSS

  • Element selector → p, h1, div
  • Class selector → .nama
  • ID selector → #nama
  • Universal selector → *

🧩 5. Properti CSS Dasar

  • color → warna teks
  • background-color → warna latar
  • font-size → ukuran teks
  • font-weight → ketebalan teks
  • text-align → posisi teks

📏 6. Box Model

  • margin → jarak luar elemen
  • padding → jarak dalam elemen
  • border → garis tepi

📦 7. Display & Layout

  • flex
  • grid

⚡ 8. Position CSS

  • relative
  • absolute
  • fixed

🌐 9. Responsive Design

CSS membuat website bisa menyesuaikan layar HP, tablet, dan laptop.

⚠️ 10. Kesimpulan

CSS sangat penting untuk membuat tampilan website menjadi menarik dan modern.

📚 11. Penjelasan CSS Lebih Lengkap

CSS digunakan untuk mengontrol tampilan website secara penuh mulai dari warna, layout, animasi, hingga responsif.

⚡ 12. Peran CSS di Dunia Web

  • Membuat UI aplikasi modern
  • Membuat website responsif
  • Mengatur layout kompleks

🧠 13. Tips CSS

  • Latihan Flexbox & Grid
  • Gunakan inspect browser
  • Sering eksperimen layout

🚀 14. Penutup

CSS adalah fondasi penting dalam web development modern.
