🖥️ Perangkat Keras (Hardware) Komputer

Hardware adalah semua perangkat fisik komputer yang dapat dilihat dan disentuh secara langsung. Hardware berfungsi untuk menjalankan input, proses, dan output pada komputer.

🧠 1. Pengertian Hardware

Hardware adalah komponen fisik komputer yang bekerja bersama software untuk menjalankan sistem komputer. Tanpa hardware, komputer tidak bisa digunakan.

⌨️ 2. Perangkat Input (MASUKAN)

Perangkat input adalah alat untuk memasukkan data ke komputer.

  • Keyboard → memasukkan teks
  • Mouse → menggerakkan pointer
  • Scanner → memindai dokumen
  • Microphone → input suara
  • Webcam → input video
  • Joystick → kontrol game

📌 Soal kuis: Keyboard, mouse, scanner = perangkat input

🖥️ 3. Perangkat Output (KELUARAN)

Perangkat output adalah alat untuk menampilkan hasil proses komputer.

  • Monitor → menampilkan gambar
  • Printer → mencetak dokumen
  • Speaker → menghasilkan suara
  • Projector → menampilkan layar besar

⚙️ 4. CPU (CENTRAL PROCESSING UNIT)

CPU adalah otak komputer yang mengolah semua data dan instruksi. CPU menentukan kecepatan komputer dalam memproses data.

💾 5. MEMORY & PENYIMPANAN

RAM (Random Access Memory)

  • Penyimpanan sementara
  • Data hilang saat komputer mati

ROM (Read Only Memory)

  • Penyimpanan tetap
  • Data tidak mudah hilang

HDD / SSD

  • Penyimpanan utama
  • SSD lebih cepat dari HDD

Flashdisk

  • Penyimpanan portable
  • Bisa dibawa ke mana saja

🔌 6. PERANGKAT PENDUKUNG (HARDWARE TAMBAHAN)

  • Motherboard → penghubung semua komponen
  • Power Supply → sumber listrik komputer
  • GPU → mengolah grafis / gambar
  • LAN Card → menghubungkan jaringan
  • Heatsink → pendingin CPU
  • UPS → cadangan listrik saat mati
  • Perangkat hybrid → touchscreen (gabungan input & output)

⚡ 7. KECEPATAN & SISTEM KOMPONEN

  • Hz → satuan kecepatan prosesor
  • CPU lebih cepat = komputer lebih cepat
  • Cache memory → mempercepat proses data

🌡️ 8. PERAWATAN HARDWARE

  • Fan → mencegah overheating
  • Heatsink → menyerap panas CPU

🔄 9. PROSES BOOTING

  • POST → pengecekan saat komputer dinyalakan
  • BIOS → tersimpan di ROM
  • Booting = proses menyalakan komputer

🌐 10. HARDWARE JARINGAN

  • NIC → menghubungkan komputer ke jaringan
  • Router → penghubung jaringan
  • Switch → penghubung perangkat
  • Modem → koneksi internet

🧩 11. KONSEP PENTING SESUAI KUIS

  • GPU = grafis
  • UPS = cadangan listrik
  • SSD = lebih cepat dari HDD
  • ROM = penyimpanan tetap
  • RAM = sementara
  • Hardware = perangkat fisik

📌 12. KESIMPULAN

Hardware adalah seluruh perangkat keras komputer yang bekerja bersama software dan brainware untuk menjalankan sistem komputer.

Tanpa hardware, komputer tidak bisa berfungsi karena semua proses input, output, dan proses bergantung pada perangkat fisik.
