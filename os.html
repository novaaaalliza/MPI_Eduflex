💻 Sistem Operasi (Operating System / OS)

Sistem Operasi adalah perangkat lunak utama yang mengatur seluruh kerja komputer. OS menjadi penghubung antara pengguna (user) dengan hardware komputer.

🧠 1. Pengertian Sistem Operasi

Sistem Operasi (OS) adalah software yang mengelola semua perangkat keras dan perangkat lunak pada komputer agar bisa berjalan dengan baik. Tanpa sistem operasi, komputer tidak dapat digunakan.

  • OS = pengatur utama komputer
  • OS = penghubung user dan hardware

⚙️ 2. Fungsi Sistem Operasi

  • Mengatur proses kerja komputer
  • Mengelola memori (RAM)
  • Mengatur file dan folder
  • Mengontrol perangkat keras (keyboard, mouse, printer)
  • Menjalankan aplikasi
  • Mengatur keamanan sistem

🧾 3. Contoh Sistem Operasi

  • Windows
  • Linux
  • MacOS
  • Android
  • iOS

📌 Soal kuis: Windows, Linux, Android = contoh sistem operasi

🧩 4. Komponen Sistem Operasi

1. Kernel

  • Inti dari sistem operasi
  • Mengatur komunikasi hardware

2. User Interface (UI)

  • Tampilan yang digunakan user
  • Contoh: desktop, icon, menu

3. System Utilities

  • Tools tambahan untuk sistem
  • Contoh: antivirus, file manager

🖥️ 5. Jenis Sistem Operasi

  • Single User → 1 pengguna
  • Multi User → banyak pengguna
  • Single Task → 1 tugas
  • Multitasking → banyak tugas sekaligus
  • Real Time OS → respon cepat (industri, mesin)

📂 6. Manajemen File

Sistem operasi mengatur file agar mudah disimpan dan dicari.

  • Folder = tempat menyimpan file
  • File = data atau dokumen
  • Extension = .docx, .jpg, .pdf

🧠 7. Manajemen Memori

  • OS mengatur penggunaan RAM
  • Mencegah komputer crash
  • Membagi memori untuk aplikasi

📌 RAM digunakan sementara saat program berjalan

⚡ 8. Manajemen Proses

  • Mengatur program yang berjalan
  • Menutup aplikasi yang error
  • Mengatur prioritas aplikasi

📌 Contoh: membuka Chrome + Spotify + Word sekaligus

🔐 9. Keamanan Sistem Operasi

  • Password login
  • User account
  • Firewall
  • Antivirus

OS menjaga agar sistem tidak mudah rusak atau diserang virus.

🌐 10. Fungsi OS dalam Jaringan

  • Menghubungkan komputer ke internet
  • Mengatur koneksi WiFi
  • Berbagi file dalam jaringan

🧩 11. CONTOH JAWABAN KUIS

  • Sistem Operasi = pengatur komputer
  • Windows = contoh OS
  • Kernel = inti OS
  • Multitasking = banyak tugas sekaligus
  • RAM = memori sementara

📌 12. KESIMPULAN

Sistem Operasi adalah software penting yang mengatur semua kerja komputer, mulai dari hardware, software, memori, hingga aplikasi agar komputer bisa berjalan dengan baik.

Tanpa sistem operasi, komputer tidak dapat digunakan karena tidak ada penghubung antara pengguna dan perangkat keras.
