🌐 INTERNET & JARINGAN DASAR

Internet adalah jaringan global yang menghubungkan jutaan komputer di seluruh dunia untuk bertukar informasi. Dengan internet, kita bisa mengakses website, mengirim email, dan berkomunikasi secara online.

🧠 1. Pengertian Internet

Internet adalah singkatan dari “Interconnected Network”, yaitu kumpulan jaringan komputer yang saling terhubung secara global. Internet tidak memiliki pusat, tetapi tersebar di seluruh dunia.

🌍 2. Fungsi Internet

  • Mencari informasi
  • Komunikasi (chat, email, video call)
  • Hiburan (YouTube, game online)
  • Pendidikan (e-learning)
  • Bisnis online

🔎 3. Web dan Browser

  • WWW (World Wide Web) → kumpulan halaman web
  • Browser → aplikasi untuk membuka website

Contoh browser:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

📩 4. Email

Email adalah surat elektronik yang digunakan untuk mengirim pesan melalui internet.

  • Contoh: Gmail, Yahoo Mail
  • Digunakan untuk komunikasi formal dan pekerjaan

⬇️ 5. Download & Upload

  • Download → mengambil data dari internet ke perangkat
  • Upload → mengirim data ke internet

📡 6. Komponen Internet

  • Modem → penghubung internet
  • Router → membagi koneksi internet
  • Server → penyedia data
  • Client → pengguna internet

⚡ 7. Teknologi Jaringan

  • WiFi → internet tanpa kabel
  • LAN → jaringan lokal
  • WAN → jaringan luas

🔐 8. Keamanan Internet (Dasar)

  • Gunakan password kuat
  • Hindari phishing
  • Gunakan website HTTPS

🧩 9. Konsep Penting di Kuis

  • Internet = jaringan global
  • WWW = kumpulan website
  • Browser = alat akses internet
  • Download = ambil data
  • Upload = kirim data
  • Google = mesin pencari

⚠️ 10. Kesimpulan

Internet adalah teknologi penting yang menghubungkan seluruh dunia. Dengan internet, informasi bisa diakses dengan cepat dan mudah dari mana saja.

🚀 Siap ke Kuis

Jika sudah memahami materi ini, kamu bisa lanjut ke kuis Internet untuk menguji pemahaman.
