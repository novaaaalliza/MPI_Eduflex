📘 Pengenalan Komputer

Komputer adalah perangkat elektronik yang digunakan untuk mengolah data menjadi informasi. Komputer bekerja dengan cara menerima input, memproses data, menyimpan data, lalu menghasilkan output. Komputer tidak bisa bekerja tanpa instruksi manusia.

💻 1. Pengertian Komputer

Komputer berasal dari kata “compute” yang berarti menghitung. Komputer adalah alat elektronik yang dapat menjalankan perintah secara otomatis berdasarkan program.

📌 Komputer modern tidak hanya untuk menghitung, tetapi juga untuk: - komunikasi - hiburan - pendidikan - bisnis

🧠 2. Komponen Sistem Komputer

✔ Hardware (Perangkat Keras)

Hardware adalah bagian fisik komputer yang bisa dilihat dan disentuh.

  • Keyboard → input teks
  • Mouse → menggerakkan pointer
  • Monitor → menampilkan output
  • Printer → mencetak dokumen
  • Speaker → menghasilkan suara

✔ Software (Perangkat Lunak)

Software adalah program yang berjalan di komputer.

  • Sistem Operasi: Windows, Linux
  • Aplikasi: Word, Excel, Chrome
  • Utility: Antivirus

✔ Brainware (Pengguna)

Brainware adalah manusia yang mengoperasikan komputer.

⌨️ 3. Perangkat Input

Perangkat input adalah alat untuk memasukkan data ke komputer.

  • Keyboard
  • Mouse
  • Scanner
  • Microphone
  • Webcam

📌 Contoh soal kuis: Mouse, keyboard, scanner = perangkat input

🖥️ 4. Perangkat Output

Perangkat output adalah alat untuk menampilkan hasil proses komputer.

  • Monitor → menampilkan gambar
  • Printer → mencetak
  • Speaker → suara
  • Projector → tampilan layar besar

⚙️ 5. CPU (Central Processing Unit)

CPU disebut sebagai “otak komputer” karena bertugas mengolah semua data dan instruksi.

📌 CPU menentukan kecepatan komputer dalam memproses data.

💾 6. Memori dan Penyimpanan

  • RAM → penyimpanan sementara (data hilang saat mati)
  • ROM → penyimpanan permanen
  • HDD / SSD → penyimpanan utama
  • Flashdisk → penyimpanan portable

🔄 7. Cara Kerja Komputer (IPO)

Input → Process → Output

  • Input = data masuk (keyboard, mouse)
  • Process = diolah CPU
  • Output = hasil (monitor, printer)

🧩 8. Jenis-Jenis Komputer

  • Desktop
  • Laptop
  • Server
  • Supercomputer
  • Mobile device (HP / tablet)

🌐 9. Fungsi Komputer

  • Mengolah data
  • Menyimpan data
  • Mengirim informasi
  • Menjalankan program

📌 10. Konsep Penting (sesuai soal kuis)

  • CPU = otak komputer
  • Mouse = perangkat input
  • Monitor = perangkat output
  • Software = program komputer
  • Hardware = perangkat fisik
  • Brainware = pengguna

📌 11. Kesimpulan

Komputer adalah sistem yang terdiri dari hardware, software, dan brainware yang bekerja bersama untuk mengolah data menjadi informasi.

Dengan memahami dasar komputer, kita bisa lebih mudah mempelajari teknologi, pemrograman, dan jaringan komputer.

Komputer sudah menjadi bagian penting dalam kehidupan modern, terutama dalam pendidikan, bisnis, dan komunikasi.
