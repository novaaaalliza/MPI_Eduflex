🌐 HTML DASAR (LENGKAP)

HTML (HyperText Markup Language) adalah bahasa dasar untuk membuat struktur halaman website. HTML digunakan untuk menampilkan teks, gambar, link, tabel, form, dan elemen web lainnya.

🧠 1. Pengertian HTML

HTML adalah bahasa markup yang digunakan untuk membangun struktur halaman web. HTML tidak bisa melakukan logika seperti JavaScript, hanya untuk struktur.

🏗️ 2. Struktur Dasar HTML

  • <!DOCTYPE html> → deklarasi HTML5
  • <html> → elemen utama
  • <head> → informasi website (judul, meta, CSS)
  • <body> → isi tampilan website

Contoh struktur:

<html>
  <head>
    <title>Judul</title>
  </head>
  <body>
    <p>Hello World</p>
  </body>
</html>

📄 3. Tag HTML Dasar

  • <h1> sampai <h6> → judul
  • <p> → paragraf
  • <br> → pindah baris
  • <hr> → garis horizontal
  • <!-- komentar --> → catatan di kode

🔗 4. Link dan Gambar

  • <a href=""> → membuat link
  • <img src=""> → menampilkan gambar

Contoh:

<a href="https://google.com">Google</a>
<img src="gambar.jpg">

📋 5. List HTML

  • Unordered List → <ul>
  • Ordered List → <ol>
  • List item → <li>
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

📦 6. Div & Container

<div> digunakan sebagai wadah untuk mengelompokkan elemen HTML.

📊 7. Tabel HTML

  • <table> → tabel
  • <tr> → baris
  • <td> → kolom
  • <th> → header tabel

Contoh:

<table>
  <tr>
    <th>Nama</th>
  </tr>
</table>

📝 8. Form HTML

  • <form> → form input
  • <input> → kolom input
  • <button> → tombol

⚙️ 9. Tag Penting di Kuis

  • <h1> → judul besar
  • <p> → paragraf
  • <a> → link
  • <img> → gambar
  • <br> → baris baru
  • <hr> → garis
  • <div> → container
  • <table> → tabel
  • <form> → input form

⚡ 10. Atribut HTML

  • href → alamat link
  • src → sumber gambar
  • alt → teks gambar
  • id → identitas unik
  • class → kelompok elemen

🧩 11. Konsep Penting

  • HTML = struktur website
  • CSS = tampilan
  • JavaScript = fungsi
  • Tag harus selalu dibuka & ditutup

⚠️ 12. Kesimpulan

HTML adalah dasar semua website. Semua halaman web dimulai dari HTML sebelum diberi desain (CSS) dan fungsi (JavaScript).

Semakin paham HTML, semakin mudah membuat website profesional.

🚀 Siap ke Kuis

Kalau sudah paham, lanjut ke kuis HTML untuk menguji kemampuanmu.
