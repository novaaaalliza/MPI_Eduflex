🗄️ Materi Database (Basis Data)

Database adalah kumpulan data yang tersimpan secara sistematis dan dapat diolah menggunakan komputer untuk menghasilkan informasi.

🧠 1. Pengertian Database

Database atau basis data adalah kumpulan data yang saling berhubungan dan disimpan secara terstruktur di dalam komputer sehingga mudah diakses, dikelola, dan diperbarui.

🎯 2. Fungsi Database

  • Menyimpan data dengan rapi
  • Mempermudah pencarian data
  • Menghindari duplikasi data
  • Mempercepat pengolahan data
  • Meningkatkan keamanan data

🏗️ 3. Komponen Database

  • Data → informasi yang disimpan
  • Tabel → tempat data disusun
  • Field → kolom dalam tabel
  • Record → baris data
  • Database Management System (DBMS)

💻 4. DBMS (Database Management System)

DBMS adalah software untuk mengelola database.

  • MySQL
  • MariaDB
  • PostgreSQL
  • Oracle
  • SQL Server

🧾 5. SQL (Structured Query Language)

SQL adalah bahasa yang digunakan untuk mengakses dan mengelola database.

  • SELECT → mengambil data
  • INSERT → menambah data
  • UPDATE → mengubah data
  • DELETE → menghapus data

🗂️ 6. Jenis Database

  • Relasional (tabel)
  • Hierarki
  • Network
  • NoSQL (MongoDB, dll)

🔑 7. Primary Key & Foreign Key

  • Primary Key → identitas unik data
  • Foreign Key → penghubung antar tabel

📊 8. ERD (Entity Relationship Diagram)

ERD adalah diagram yang menunjukkan hubungan antar tabel dalam database.

  • Entity → objek (contoh: siswa, guru)
  • Attribute → karakteristik data
  • Relationship → hubungan antar entity

⚙️ 9. Operasi Database

  • Create → membuat database
  • Read → membaca data
  • Update → mengubah data
  • Delete → menghapus data

🔐 10. Keamanan Database

  • Password database
  • Hak akses user
  • Backup data
  • Enkripsi data

📡 11. Contoh Penggunaan Database

  • Data siswa sekolah
  • Data nilai
  • Data pegawai
  • Data transaksi toko

🧠 12. KONSEP KUIS PENTING

✔ Database = kumpulan data ✔ SQL = bahasa database ✔ SELECT = ambil data ✔ Primary Key = unik ✔ Record = baris ✔ Field = kolom ✔ DBMS = pengelola database ✔ CRUD = operasi dasar database

📌 13. KESIMPULAN

Database adalah sistem penting untuk menyimpan dan mengelola data secara terstruktur. Dengan database, data menjadi lebih rapi, aman, dan mudah diakses.

Database digunakan hampir di semua sistem modern seperti sekolah, bank, toko online, dan aplikasi digital lainnya.
