⚡ JavaScript Dasar

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website menjadi interaktif, dinamis, dan responsif terhadap pengguna. JavaScript bekerja bersama HTML dan CSS untuk membangun halaman web modern.

🧠 1. Pengertian JavaScript

JavaScript adalah bahasa pemrograman yang berjalan di browser (client-side scripting). Artinya, JavaScript dijalankan langsung di browser pengguna tanpa perlu server tambahan.

JavaScript digunakan untuk menambahkan logika dan interaksi pada website seperti tombol, animasi, validasi form, dan game sederhana.

  • HTML = struktur
  • CSS = tampilan
  • JavaScript = interaksi

🌐 2. Fungsi JavaScript

  • Membuat website interaktif
  • Validasi form login / registrasi
  • Membuat game sederhana
  • Mengubah isi halaman tanpa reload
  • Membuat animasi dan efek
  • Mengontrol elemen HTML

🧾 3. Cara Penulisan JavaScript

Contoh:

<script>
alert("Halo Dunia!");
</script>

Penjelasan:

  • <script> = tempat menulis JavaScript di HTML
  • alert() = menampilkan pesan popup

🧩 4. Variabel

let nama = "Budi";
const umur = 17;

Penjelasan:

  • let = variabel yang bisa diubah
  • const = variabel tetap (tidak bisa diubah)
  • var = cara lama (jarang digunakan)
  • variabel = tempat menyimpan data

🔢 5. Tipe Data

  • String = teks ("Halo")
  • Number = angka (10, 20, 3.5)
  • Boolean = true / false
  • Array = kumpulan data
  • Object = data yang lebih kompleks
  • Undefined = belum memiliki nilai

⚙️ 6. Operator

  • + = penjumlahan
  • - = pengurangan
  • * = perkalian
  • / = pembagian
  • % = sisa bagi

Contoh:

10 + 5 = 15
20 - 3 = 17

🔁 7. IF ELSE (Percabangan)

if (nilai > 75) {
  console.log("Lulus");
} else {
  console.log("Tidak Lulus");
}

Penjelasan:

  • IF = jika kondisi benar
  • ELSE = jika kondisi salah
  • digunakan untuk pengambilan keputusan

🔄 8. Loop (Perulangan)

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

Penjelasan:

  • for = perulangan
  • i++ = bertambah 1 setiap putaran
  • digunakan untuk mengulang proses

🖱️ 9. Event (Interaksi User)

Penjelasan:

  • onclick = saat tombol diklik
  • onmouseover = saat mouse diarahkan
  • event = aksi pengguna

🧠 10. DOM (Document Object Model)

document.getElementById("judul").innerHTML = "Halo!";

Penjelasan:

  • DOM = cara JavaScript mengubah HTML
  • getElementById = mengambil elemen HTML
  • innerHTML = mengubah isi teks

🧩 11. FUNCTION

function sapa() {
  alert("Halo!");
}

Penjelasan:

  • function = kumpulan perintah
  • bisa dipanggil berkali-kali

📌 12. CONTOH KESIMPULAN KUIS

  • JavaScript = bahasa pemrograman web interaktif
  • alert = popup pesan
  • let = variabel bisa berubah
  • const = variabel tetap
  • DOM = manipulasi HTML
  • function = kumpulan perintah

📌 13. KESIMPULAN

JavaScript adalah bahasa pemrograman penting dalam web development yang digunakan untuk membuat website menjadi interaktif, dinamis, dan responsif.

Dengan JavaScript, kita bisa membuat tombol interaktif, validasi data, animasi, game sederhana, dan aplikasi web modern.
