🔐 Materi Keamanan Jaringan Komputer

Keamanan jaringan adalah cara untuk melindungi sistem jaringan komputer dari ancaman, gangguan, dan akses tidak sah.

🧠 1. Pengertian Keamanan Jaringan

Keamanan jaringan adalah sistem yang digunakan untuk melindungi data, perangkat, dan jaringan dari serangan seperti virus, hacking, dan pencurian data.

📌 Kuis: Keamanan jaringan = melindungi data & sistem dari ancaman

🎯 2. Tujuan Keamanan Jaringan

  • Menjaga kerahasiaan data (confidentiality)
  • Menjaga integritas data (tidak diubah sembarangan)
  • Menjaga ketersediaan sistem (availability)
  • Melindungi dari hacker dan malware

⚠️ 3. Ancaman dalam Jaringan

  • Virus → merusak file
  • Worm → menyebar sendiri
  • Trojan → menyamar sebagai aplikasi baik
  • Hacking → akses ilegal
  • Phishing → pencurian data
  • Spyware → memata-matai pengguna
📌 Kuis: Phishing = penipuan untuk mencuri data

🛡️ 4. Teknik Keamanan Jaringan

  • Firewall → penyaring akses masuk/keluar
  • Antivirus → melindungi dari virus
  • Enkripsi → mengubah data jadi kode
  • Authentication → login dengan password
  • Backup data → cadangan data

🔑 5. Password & Autentikasi

Password digunakan untuk memastikan hanya pengguna tertentu yang bisa mengakses sistem.

  • Password kuat = kombinasi huruf, angka, simbol
  • Two-factor authentication (2FA)
📌 Kuis: Password = keamanan akses login

🌐 6. Firewall

Firewall adalah sistem yang menyaring data masuk dan keluar dari jaringan.

  • Memblokir akses berbahaya
  • Mengizinkan akses aman

🔐 7. Enkripsi Data

Enkripsi adalah proses mengubah data menjadi kode agar tidak mudah dibaca orang lain.

📌 Kuis: Enkripsi = mengubah data menjadi kode rahasia

🧩 8. Jenis Serangan Siber

  • DDoS → membuat server down
  • Malware → software berbahaya
  • SQL Injection → menyerang database
  • Brute force → menebak password

🖥️ 9. Cara Mencegah Serangan

  • Gunakan password kuat
  • Update sistem secara rutin
  • Gunakan antivirus
  • Jangan klik link mencurigakan
  • Gunakan firewall

📡 10. Keamanan di Jaringan WiFi

  • Gunakan password WiFi yang kuat
  • Gunakan WPA2/WPA3
  • Hindari WiFi publik untuk login penting

🧠 11. KONSEP KUIS PENTING

✔ Firewall = penyaring jaringan ✔ Virus = merusak file ✔ Phishing = pencurian data ✔ Enkripsi = pengkodean data ✔ Password = akses keamanan

📌 12. KESIMPULAN

Keamanan jaringan sangat penting untuk melindungi data dan sistem dari ancaman. Tanpa keamanan, data bisa dicuri, dirusak, atau disalahgunakan oleh pihak yang tidak bertanggung jawab.

Dengan menggunakan firewall, antivirus, enkripsi, dan password yang kuat, sistem jaringan menjadi lebih aman dan terpercaya.
