🌐 Materi Jaringan Komputer

🌐 Jaringan Komputer (Computer Network)

Jaringan komputer adalah hubungan dua atau lebih perangkat komputer yang saling terhubung untuk berbagi data, informasi, dan sumber daya seperti printer, file, dan internet.

💻 1. Pengertian Jaringan Komputer

Jaringan komputer adalah sistem yang memungkinkan komputer saling terhubung dan berkomunikasi menggunakan media kabel atau nirkabel.

📡 2. Tujuan Jaringan Komputer

  • Berbagi data dengan cepat
  • Berbagi perangkat keras (printer, scanner)
  • Mempermudah komunikasi
  • Meningkatkan efisiensi kerja
  • Menghemat biaya operasional

🌍 3. Jenis-Jenis Jaringan

  • LAN → Local Area Network (ruangan/sekolah)
  • MAN → Metropolitan Area Network (antar kota)
  • WAN → Wide Area Network (antar negara)
  • PAN → Personal Area Network (HP, headset, Bluetooth)

📌 Internet = WAN terbesar di dunia

📶 4. Perangkat Jaringan

  • Router → menghubungkan internet antar jaringan
  • Switch → menghubungkan banyak komputer dalam satu jaringan
  • Hub → alat pembagi sinyal (versi lama switch)
  • Modem → mengubah sinyal digital ke internet
  • Access Point → pemancar WiFi
  • LAN Card → kartu jaringan komputer

🔌 5. Media Transmisi

  • Kabel UTP → kabel jaringan LAN
  • Fiber Optic → sangat cepat dan stabil
  • Kabel Coaxial → jaringan lama
  • Wireless → tanpa kabel (WiFi)

📡 6. Topologi Jaringan

  • Bus → satu kabel utama
  • Star → pusat ke semua perangkat
  • Ring → melingkar
  • Mesh → semua saling terhubung
  • Tree → bentuk seperti pohon (gabungan star)

📌 Star paling sering dipakai di sekolah

🌐 7. Internet

Internet adalah jaringan global yang menghubungkan seluruh dunia menggunakan protokol komunikasi.

  • Akses informasi
  • Komunikasi (chat, email, video call)
  • Pembelajaran online
  • Hiburan dan media sosial

🧠 8. IP Address

IP Address adalah alamat unik untuk setiap perangkat dalam jaringan.

  • Contoh: 192.168.0.1
  • Setiap perangkat harus berbeda IP
  • Digunakan untuk identifikasi jaringan

🔐 9. Keamanan Jaringan

  • Password WiFi
  • Firewall → pelindung sistem dari ancaman
  • Antivirus
  • Enkripsi data

⚙️ 10. Protokol Jaringan

  • HTTP → untuk website
  • HTTPS → versi aman HTTP
  • FTP → transfer file
  • TCP/IP → aturan komunikasi internet

📌 11. Konsep Kuis Penting

  • LAN = jaringan kecil
  • WAN = jaringan luas
  • PAN = jaringan pribadi
  • Router = penghubung internet
  • Switch = penghubung komputer
  • IP Address = alamat perangkat
  • Firewall = keamanan jaringan

⚡ 12. Kelebihan Jaringan

  • Transfer data cepat
  • Komunikasi mudah
  • Hemat biaya
  • Efisiensi kerja tinggi

⚠️ 13. Kekurangan Jaringan

  • Rentan serangan virus
  • Ketergantungan pada internet
  • Gangguan jaringan bisa menghambat kerja

📌 14. Kesimpulan

Jaringan komputer adalah sistem penting yang menghubungkan perangkat agar dapat saling bertukar data dengan cepat dan efisien.

Dengan jaringan komputer, komunikasi, pembelajaran, dan pekerjaan menjadi lebih mudah dan terhubung secara global.
