📘 Materi Microsoft Word Dasar

📝 Microsoft Word (Pengolah Kata)

Microsoft Word adalah aplikasi pengolah kata yang digunakan untuk membuat, mengedit, menyimpan, dan mencetak dokumen seperti surat, laporan, makalah, dan tugas sekolah.

💻 1. Pengertian Microsoft Word

Microsoft Word merupakan software dari Microsoft Office yang digunakan untuk mengolah teks secara digital. Word memudahkan pekerjaan mengetik dibandingkan menulis manual di kertas.

📄 2. Fungsi Microsoft Word

  • Membuat dokumen teks
  • Mengedit tulisan dengan mudah
  • Membuat surat resmi dan laporan
  • Menyisipkan gambar, tabel, dan grafik
  • Mencetak dokumen (print)

🧠 3. Menu Utama Microsoft Word

  • File → membuat, membuka, menyimpan dokumen
  • Home → pengaturan teks (font, paragraf)
  • Insert → menambahkan gambar, tabel, shape
  • Design → tema dokumen
  • Layout → margin, ukuran kertas
  • Review → pengecekan ejaan dan komentar
  • View → tampilan dokumen

⌨️ 4. Toolbar Dasar

  • Bold (B) → menebalkan teks
  • Italic (I) → memiringkan teks
  • Underline (U) → menggarisbawahi teks
  • Font → jenis huruf
  • Size → ukuran huruf
  • Color → warna huruf

📌 Kuis: B = Bold, I = Italic, U = Underline

📊 5. Tabel dalam Word

Tabel digunakan untuk menyusun data agar lebih rapi dan mudah dibaca.

  • Insert Table → membuat tabel
  • Add Row/Column → menambah baris/kolom
  • Merge Cells → menggabungkan sel
  • Border → garis tabel

🖼️ 6. Gambar & Objek

  • Insert Picture → memasukkan gambar
  • Resize → mengubah ukuran gambar
  • Position → mengatur letak gambar
  • Wrap Text → mengatur teks di sekitar gambar

📌 7. Shortcut Penting

  • Ctrl + C → copy
  • Ctrl + V → paste
  • Ctrl + X → cut
  • Ctrl + S → save
  • Ctrl + P → print
  • Ctrl + Z → undo
  • Ctrl + Y → redo

⚙️ 8. Format Dokumen

  • Alignment → rata kiri, kanan, tengah, justify
  • Line spacing → jarak antar baris
  • Margin → batas halaman
  • Paper size → ukuran kertas

🧩 9. Konsep Kuis Penting

  • Word = pengolah kata
  • Insert = menambah elemen
  • Save = menyimpan dokumen
  • Print = mencetak dokumen
  • Home = pengaturan teks utama

🌍 10. Kelebihan Microsoft Word

  • Mudah digunakan
  • Hasil rapi dan profesional
  • Bisa diedit kapan saja
  • Mendukung gambar dan tabel

📌 11. Kekurangan Microsoft Word

  • Berbayar (tidak gratis penuh)
  • Butuh perangkat komputer
  • File bisa rusak jika tidak disimpan

📌 12. Kesimpulan

Microsoft Word adalah aplikasi pengolah kata yang sangat penting dalam dunia pendidikan dan pekerjaan. Word membantu membuat dokumen lebih cepat, rapi, dan efisien.

Dengan menguasai Word, pengguna dapat membuat berbagai dokumen profesional seperti surat, laporan, dan makalah.
