🗄️ Materi MySQL (Database)

MySQL adalah DBMS yang digunakan untuk mengelola database menggunakan bahasa SQL.

🧠 1. Pengertian MySQL

MySQL adalah sistem manajemen basis data (DBMS) yang menggunakan SQL untuk menyimpan, mengubah, dan mengambil data dari database. MySQL bersifat open source dan banyak digunakan pada aplikasi web.

📌 Kuis: MySQL = DBMS untuk mengelola database

🎯 2. Fungsi MySQL (Diperluas)

  • Menyimpan data dalam jumlah besar
  • Mengelola data secara terstruktur
  • Mendukung aplikasi web dan mobile
  • Mengamankan data pengguna
  • Mengatur hak akses user
  • Mendukung multi-user (banyak pengguna)

⚙️ 3. Struktur MySQL

  • Database → kumpulan tabel
  • Tabel → tempat data disimpan
  • Field → kolom data
  • Record → baris data
📌 Kuis: Field = kolom, Record = baris

🧾 4. SQL Command (Lebih Lengkap)

  • CREATE DATABASE → membuat database
  • DROP DATABASE → menghapus database
  • CREATE TABLE → membuat tabel
  • ALTER TABLE → mengubah struktur tabel
  • INSERT INTO → menambah data
  • SELECT → menampilkan data
  • UPDATE → mengubah data
  • DELETE → menghapus data
📌 Kuis: SELECT = menampilkan data

💻 5. Contoh Query Penting

CREATE DATABASE sekolah;
USE sekolah;

CREATE TABLE siswa (
id INT PRIMARY KEY,
nama VARCHAR(100),
kelas VARCHAR(10)
);

INSERT INTO siswa VALUES (1, 'Budi', 'X RPL');

SELECT * FROM siswa;

🔑 6. Primary Key & Foreign Key (Diperjelas)

  • Primary Key → identitas unik (tidak boleh sama)
  • Foreign Key → penghubung antar tabel
  • Relasi penting untuk database besar
📌 Kuis: Primary Key = unik, tidak boleh sama

📊 7. CRUD (Lebih Detail)

  • Create → INSERT
  • Read → SELECT
  • Update → UPDATE
  • Delete → DELETE
📌 Kuis: CRUD = Create Read Update Delete

🧩 8. Relasi Database

  • One to One → satu ke satu
  • One to Many → satu ke banyak
  • Many to Many → banyak ke banyak

🔐 9. Keamanan MySQL

  • Username & password database
  • GRANT (hak akses user)
  • REVOKE (hapus akses user)
  • Backup database secara rutin
  • Firewall server database

⚡ 10. Kelebihan MySQL

  • Gratis dan open source
  • Ringan dan cepat
  • Mudah dipelajari
  • Cocok untuk web development
  • Stabil untuk data besar

❌ 11. Kekurangan MySQL

  • Kurang cocok untuk data super kompleks tertentu
  • Fitur enterprise lebih sedikit dibanding Oracle

🧠 12. KONSEP KUIS PENTING (RINGKAS)

✔ MySQL = DBMS ✔ SQL = bahasa database ✔ SELECT = ambil data ✔ INSERT = tambah data ✔ UPDATE = ubah data ✔ DELETE = hapus data ✔ Primary Key = unik ✔ Foreign Key = penghubung tabel ✔ CRUD = operasi database

🌍 13. Penerapan MySQL di Dunia Nyata

  • Sistem nilai sekolah
  • E-commerce (Shopee, Tokopedia)
  • Bank (data nasabah)
  • Aplikasi login & user
  • Sistem absensi

📌 14. KESIMPULAN

MySQL adalah sistem database penting yang digunakan hampir di semua aplikasi modern. Dengan MySQL, data dapat disimpan dengan aman, terstruktur, dan mudah diakses.

Pemahaman MySQL sangat penting untuk pemrograman web, sistem informasi, dan pengolahan data.
