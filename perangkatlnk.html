💻 PERANGKAT LUNAK (SOFTWARE)

Software adalah kumpulan program yang digunakan untuk menjalankan komputer dan mengolah data. Software tidak bisa disentuh karena berbentuk digital, tetapi sangat penting agar komputer dapat berfungsi.

🧠 1. Pengertian Software

Software adalah sekumpulan instruksi atau program yang memberi perintah kepada komputer agar dapat bekerja. Tanpa software, hardware tidak bisa digunakan.

⚙️ 2. Jenis-Jenis Software

1. Sistem Operasi (Operating System)

  • Windows
  • Linux
  • MacOS
  • Android

📌 Fungsi: mengatur semua hardware dan software agar bisa berjalan.

2. Software Aplikasi

  • Microsoft Word → pengolah kata
  • Microsoft Excel → pengolah angka
  • PowerPoint → presentasi
  • Google Chrome → browser

3. Software Utility

  • Antivirus
  • Disk Cleanup
  • Backup system

📌 Fungsi: membantu perawatan komputer.

🌐 3. Contoh Software Populer

  • Windows (OS paling umum)
  • Linux (open source)
  • Microsoft Office (Word, Excel, PowerPoint)
  • Adobe Photoshop (edit gambar)
  • Google Chrome (browser)

📂 4. File dan Ekstensi Software

  • .exe → program aplikasi
  • .docx → Microsoft Word
  • .xlsx → Excel
  • .pptx → PowerPoint
  • .jpg/.png → gambar

🧩 5. Konsep Penting di Kuis

  • Software = perangkat lunak
  • Windows = sistem operasi
  • Word = pengolah kata
  • Excel = pengolah angka
  • Chrome = browser
  • Linux = OS open source
  • Driver = penghubung hardware dan OS

⚡ 6. Cara Kerja Software

Software bekerja dengan memberikan instruksi ke hardware melalui sistem operasi. Contoh: User klik → OS memproses → hardware menjalankan perintah.

📌 7. Kelebihan Software

  • Membuat komputer bisa digunakan
  • Mudah diperbarui
  • Bisa disesuaikan kebutuhan

⚠️ 8. Kesimpulan

Software adalah bagian penting komputer yang berisi program untuk mengatur dan menjalankan hardware. Tanpa software, komputer hanya menjadi benda mati.
